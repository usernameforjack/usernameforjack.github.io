1) uname -a

The comand uname -a reveals the linux system information



2) top

Displays and manages the top running processes



3) who

Finds who is currently logged into the system



4) ls -al

List all files in a long listing (detailed) format



5) mkdir directory

Create a directory

6) rm file

Remove (delete) file



7) cp file1 file2

Copy file1 to file2



8) cat file

View the contents of file



9) mv file1 file2

Rename or move file1 to file2. If file2 is an existing directory, move file1 into directory file2



10) locate name

Find files and directories by name
